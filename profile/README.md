## Overview

![4 Seasons Event](https://photos.4seasons.dance/s/resynced/photos/9b863895-0f14-4d7b-a595-3b2dd33b894a)

4 Seasons is an annual mixed-reality event in the Netherlands focused on merging VR culture with real-world gatherings.  
This GitHub organisation contains internal code, infrastructure, automations and future public software for the event ecosystem.

Current repositories are private. Public release of selected projects may happen later depending on maturity, security review and licensing.

---

## Scope of This Organization

Repositories within this organization may include:

- Web applications and landing pages
- Ticketing, registration and access control logic
- Backend services and REST or GraphQL API layers
- Internal operations tools and event automation
- Discord and VR integration utilities
- DevOps, IaC, deployment and build pipelines

Each repository is expected to provide its own documentation and architectural explanation.
